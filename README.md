# Features
- 主机管理：基于主机的资源信息，主机的用途对归属关系的管理
- 组织架构管理：基于业务需求的组织架构管理
- 权限管理：基于成员组的权限管理
- 访问策略：基于安全组定义访问规则
- 版本控制：基于CI/CD提供透明的流程，反馈各阶段的详细信息
- 事件发现与推送：基于回调方式对事件的发现与推送
- 审计：对用户敏感操作提供留存审计记录的功能
